Settings

Change User Interface language to display the help content in your own language. This does not affect the Pali script setting below.

Pali language can be written using many different character sets. By changing the the pali script in the settings all the pali text displayed on the site will be changed.

Dark Mode allows you to read the text for a long time without much eye strain specially in the low light situations and at night time.

Single - Tabs - Columns - display multiple text files either side-by-side in split screen mode or as tabs. For small screen devices disabling multi files by selecting the single mode is recommended.

Text size of the application can be changed by "Text Size" setting.

Variant reading or footnotes can be displayed in various different ways by changing the "Footnotes Format" setting

Page breaks in various different print editions of tipitaka can be displayed by changing the "Page Tags Format" setting

Title Bar

Title bar allows you to navigate around the application.

  1. Search Bar - Search term(s) for searches should be typed here
  2. Search Type Icon - Rotate through different search types by clicking this icon.
  3. Text - Jump to Text View - your opened text files will be displayed
  4. Settings - Access the tipitaka.app settings
  5. Menu bar - Access miscellaneous functionality of this application.

Searching

Tipitaka.app supports three different search modes

  1. Title Search
  2. Full Text Search
  3. Dictionary Search

Title Search (TS)

  • This is the default search mode
  • Searches through the sutta names
  • Useful in quickly finding a specific sutta by name

How to Use

  • Simply start typing the sutta name in the search bar.
  • Search can be limited to a set of book by using the filter
  • Wildcard searches using the wildcards % and _ "Appa%me" are allowed
  • At any time to go back to the search results, simply click on the search bar.

Full Text Search (FTS)

Allows you to search the sutta content of whole tipitaka (mula), atta, tika and anya.

How to Use

  • First FTS search mode need to be selected by clicking on the search button next to the search bar until the icon appears.
  • Then type the search term(s) that you need to search.
  • Wildcard characters % (any number of characters) and _ (one character) are allowed in each of the search terms.
  • A list of matching words, the number of occurences of each word, and the number of files the word appears will be displayed.
  • Clicking on each word will show the list of text files and the number of occurences in each of those text file.
  • Clicking on one of the files will take you to that file where the word will be highlighted.
  • Search can be limited to a set of book by using the filter

Example Search Phrases

  • rūpaṁ aniccaṁ
  • Appamādo amatapadaṁ, pamādo maccuno padaṁ
  • %pamādo amata%

Search Options

  1. Sort By Book - Sort the list of text files in the order of Mula, Atta, Tika instead of the number of occurances.
  2. Exact Word - if this is disabled % will be added to the end of each term in the search phrase
  3. Exact Phrase - If this is enabled the words matching the search terms should appear one after the other in the text
  4. Match Order - The order of the search terms should be maintained by the words in the text
  5. Word Distance - Words matching the search terms should all occur within an interval of this many words in the text.

Dictionary Search

Pali dictionaries of various languages has been included with tipitaka.app so that the meanings of the pali words in tipitaka can be easily looked up.

First select the dictionary search mode by clicking on the search button. Then select the dictionaries that you want to be looked up by clicking the "Select Dictionaries" button.

By clicking on any pali word will launch the Pali Analysis window, where more search options are available for that word.

If the pali word does not have any exact matches in any of the selected dictionaries, a single vowel (or anusawara) character will be removed from the end of the pali word and tried again automatically. This is called fuzzy matching.

If even the fuzzy matching does not return any results, you can manually edit the word in the search bar until some matches appear.

Warning: Always check that the meanings looked up are for the same or related pali words to the original word.

Bookmarks

You can save bookmarks by clicking on the star icon next to the sutta names (titles) in either the search results view or text view.

Saved bookmarks can be accessed by clicking the Star icon in the menu bar. Number of saved bookmarks might also be displayed in the menu bar.

If you are using the website, clearing the browser cache will clear the bookmarks.

Tree - Sutta Hierarchy

The sutta tree shows the hierachical listing of suttas based on the sutta location.

Each tree branch can be opened/closed by clicking on the branch name.

Clicking on the tree leaves will open the corresponding text file for viewing

On small screen devices the tree will not be displayed by default and has to be opened by clicking on the tree icon in the title bar.

Text View

The source text has been split into 2662 files/sections. Each can be accessed by clicking the correspoinding tree leaf in the tree view, or by clicking on search results.

The text belonging to each of the titles in each file will be collapsed by default. All sections can be expanded/collapsed by clicking on the top most title. Individual sections can also be expanded/collapsed by clicking on the corresponding titles

By clicking on the M/A/T buttons on the top left corner of the text files, you can open the corresponding Mula/Atta/Tika file.

Direct Links

Direct links to specific sections of the text files can be copied by clicking on the share(arrow) icon next to the titles or paragraph numbers displayed in the text file.

The direct links have the following format. https://tipitaka.app/?a=aa1-41-ro. It is composed of 3 parts.

  1. aa1 - refers is the file name
  2. 41 - refers to the line number in that file
  3. ro - refers to the currently selected pali display script

When clicked these links will open the full tipitaka.app application. Then the text file will be opened. Finally the section/line in that file will be expanded and highlighted.
